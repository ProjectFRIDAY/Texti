구름IDE - GITHUB 연결
-----
**구름 Git 탭에서 원격 저장소로 Push시 오류남!** -> 터미널에 직접 명령어를 입력해서 push해주세요.
+ 모든 변경 사항을 Push하는 방법 (main 브런치)
```
git add --all
git commit -m "커밋 내용"
git push -u origin main
```

구름IDE에서의 실행
-----
구름IDE 화면에서 우측 상단에 있는 계정 이메일 밑에 아이콘들 중 '**▶ 버튼 (오른쪽에서 두번째)**'에 마우스를 올리고, 원하는 포트를 선택하면 바로 실행됩니다.
